# mythicalOS

Design and engineering for the mythicalOS family.

**[mythicalos.ai](https://mythicalos.ai)** — something is being forged.

Most of the work here is not public yet. What is:

- **[mythical-design](https://github.com/mythicalOS/mythical-design)** — `@mythicalos/tokens`:
  the design-system foundation. Canonical tokens, self-hosted fonts, brand assets, and the
  design book. Apache-2.0.

More will open over time. When the products land, they will install from
**[get.mythicalos.ai](https://get.mythicalos.ai)**.
